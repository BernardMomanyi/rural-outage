Project Overview

OutageSys is a smart, responsive web-based system designed to predict, manage, and reduce power outages in rural areas by leveraging historical data, analytics, and real-time reporting. Our mission is to improve rural electricity reliability and empower communities with data-driven insights.

Core Features

  • Outage Prediction Engine: Uses statistical models to predict possible power outages and helps in early detection of high-risk substations.
  • Interactive Substation Map: Visual display of all rural substations, showing status and predicted risk level.
  • Real-Time Dashboard Analytics: Number of active substations, outage trends, critical alerts, and easy-to-read summaries.
  • CSV Data Upload: Upload past outage reports or substation logs to improve prediction accuracy.
  • Admin Management Tools: Add, edit, and delete substations; update system thresholds and prediction parameters.
  • Secure User Login: Role-based access for Admin, Technician, and Viewer.
